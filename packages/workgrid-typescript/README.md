# Workgrid Typescript

![Latest version](https://img.shields.io/npm/v/@workgrid/typescript.svg)

## Installation

```bash
$ npm install --save-dev @workgrid/typescript
```

## Usage

tsconfig.json

```json
{
  "extends": "@workgrid/typescript",
  "exclude": ["**/*.test.ts", "dist", "coverage"]
}
```
