# Workgrid Microapp

![Latest version](https://img.shields.io/npm/v/@workgrid/microapp.svg)

## Installation

```bash
$ npm install @workgrid/microapp
```

## Usage

```js
import Microapp from '@workgrid/microapp'
const microapp = new Microapp()

microapp.initialize()
const token = await microapp.getToken()
```
